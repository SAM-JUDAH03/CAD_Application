## [1.0.0] - 2026-06-05

### Added
- Initial release of CAD Application
- Main WPF drawing window with MVVM architecture
- `DrawingState` model for managing drawing session data
- `MainViewModel` for binding UI to business logic
- Helper classes: `BrushHelper`, `GeometryHelper`, `SerializationHelper`
- Integration with Extended WPF Toolkit (Xceed) v5.0.0
- Basic asset icons (add, back)
- App configuration and NuGet package setup
